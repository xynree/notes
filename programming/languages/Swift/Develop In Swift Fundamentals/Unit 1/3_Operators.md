# 1.3 - Operators

- assignment operator: `=`
- basic arithmetic: `+,-,*,/`
- If consts or vars are explicitly declared as `Double`, results will include decimals
- Compound Assignment: `+=, -=, *=, /=`
- Remainder: `%`
- Numeric Type Conversion: Can cast numeric types to others:

```swift

let x = 3
let y = 0.1415927
let pi = Double(x) + y
```
